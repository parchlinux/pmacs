# pmacs documentation

Welcome to the pmacs documentation. pmacs is an Emacs distribution built for
Parch Gnu/Linux that combines the power of Evil-mode with a curated selection
of Emacs packages. It is designed to be fast, simple, and well-integrated with
the Parch system.

- [Installation](installation.md) covers getting pmacs up and running on your
  machine, including prerequisites, setup, and package management.
- [Configuration](configuration.md) explains the configuration system,
  directory structure, and how to customize pmacs to your liking.
- [Modules](modules.md) describes the seven module categories that ship with
  pmacs, what they provide, and how they work.
- [Keybindings](keybindings.md) is your reference for the default keybindings,
  including Evil mode, Emacs mode, and Parch-specific commands.
- [Parch integration](parch-integration.md) documents how pmacs works with
  Parch Gnu/Linux tools like pacman, AUR helpers, and systemd.
- [Contributing](contributing.md) explains how to contribute to pmacs
  development, including coding standards and testing procedures.
